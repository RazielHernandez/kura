# Kura

> **Collect. Organize. Enjoy.**

Kura is an offline-first collection manager that helps you organize everything you own or want to collect in one place.

Whether you collect video games, books, trading cards, LEGO sets, movies, action figures, vinyl records, electronics, or something completely unique, Kura gives you a beautiful and flexible way to manage your collections.

---

## ✨ Features

### Version 1 (Offline)

- 📁 Create unlimited collections
- 📦 Add unlimited items
- 🖼️ Multiple photos per item
- 📝 Notes and descriptions
- 🏷️ Tags
- ⭐ Favorites
- 🔍 Fast search
- 📊 Collection statistics
- 🌙 Light & Dark Mode
- 💾 100% Offline
- 📱 Native experience

## 🎨 Design Philosophy

Kura is inspired by Apple's design principles.

The interface should be:

- Clean
- Minimal
- Fast
- Intuitive
- Content-first

Collections should feel like browsing a personal library rather than managing a database.

---

## 🌟 Vision

Our goal is to build the best personal collection manager available.

Instead of creating separate apps for books, games, movies, collectibles, or electronics, Kura provides one flexible platform where users can organize anything they care about.

Kura starts as an offline-first application focused on simplicity, performance, and privacy, while laying the foundation for future cloud synchronization, AI-powered organization, and cross-platform support.

---

## 🔒 Privacy

Version 1 stores all data locally on your device.

No account is required.

No internet connection is required.

Future cloud synchronization will always be optional.
